- **–ö–ª—é—á–µ–≤—ã–µ –≤–æ–ø—Ä–æ—Å—ã**:
  - –ö–∞–∫–∏–µ —Ç–µ–º—ã —á–∞—â–µ –≤—Å–µ–≥–æ –ø–æ—è–≤–ª—è—é—Ç—Å—è?
  - –ö–∞–∫–∏–µ —ç–º–æ—Ü–∏–∏ —Å–≤—è–∑–∞–Ω—ã —Å –∫–∞–∂–¥–æ–π —Ç–µ–º–æ–π?

### –ü–æ–¥—Ö–æ–¥ –∏ –º–µ—Ç–æ–¥—ã
1. **–¢–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ**:
   - –ò—Å–ø–æ–ª—å–∑—É–µ–º **LDA (Latent Dirichlet Allocation)** –¥–ª—è –≤—ã–¥–µ–ª–µ–Ω–∏—è —Ç–µ–º.
   - –ü–æ–¥–≥–æ—Ç–æ–≤–∏–º –¥–∞–Ω–Ω—ã–µ: —É–¥–∞–ª–∏–º —Å—Ç–æ–ø-—Å–ª–æ–≤–∞, –ø—Ä–∏–≤–µ–¥—ë–º —Ç–µ–∫—Å—Ç –∫ –ª–µ–º–º–∞–º.
2. **–°–≤—è–∑—å —Å —ç–º–æ—Ü–∏—è–º–∏**:
   - –í—ã—á–∏—Å–ª—è–µ–º —ç–º–æ—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π —Ñ–æ–Ω –¥–ª—è –∫–∞–∂–¥–æ–π —Ç–µ–º—ã.

### –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–æ–≤
- –£—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º –±–∏–±–ª–∏–æ—Ç–µ–∫–∏:
  ```bash

  ```
- –ü—Ä–∏–º–µ—Ä —Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ –∞–Ω–∞–ª–∏–∑–∞:
  ```python
  from gensim import corpora
  from gensim.models import LdaModel
  from nltk.corpus import stopwords
  from nltk.tokenize import word_tokenize

  # –ü–æ–¥–≥–æ—Ç–æ–≤–∫–∞ –¥–∞–Ω–Ω—ã—Ö
  stop_words = set(stopwords.words("russian"))
  messages = ["–ü—Ä–∏–≤–µ—Ç, –∫–∞–∫ –¥–µ–ª–∞?", "–°–µ–≥–æ–¥–Ω—è –ø—Ä–µ–∫—Ä–∞—Å–Ω–∞—è –ø–æ–≥–æ–¥–∞!"]
  texts = [[word for word in word_tokenize(msg.lower()) if word.isalnum() and word not in stop_words] for msg in messages]

  # –°–æ–∑–¥–∞–Ω–∏–µ –º–æ–¥–µ–ª–∏
  dictionary = corpora.Dictionary(texts)
  corpus = [dictionary.doc2bow(text) for text in texts]
  lda_model = LdaModel(corpus, num_topics=2, id2word=dictionary, passes=15)

  # –í—ã–≤–æ–¥ —Ç–µ–º
  topics = lda_model.print_topics()
  for topic in topics:
      print(topic)
  ```
